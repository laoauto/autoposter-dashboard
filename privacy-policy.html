AutoPost Manager · ອັບເດດຫຼ້າສຸດ: 14 ກໍລະກົດ 2026

AutoPost Manager ("ລະບົບ", "ພວກເຮົາ") ເປັນເຄື່ອງມືພາຍໃນ (internal tool) ສຳລັບຜູ້ບໍລິຫານ Facebook Page ຂອງຕົນເອງ ໃຊ້ເພື່ອຊ່ວຍໂພດຮູບ ແລະ ຂໍ້ຄວາມໄປຍັງ Facebook Page ທີ່ຜູ້ໃຊ້ເປັນເຈົ້າຂອງ/ຜູ້ບໍລິຫານໂດຍກົງເທົ່ານັ້ນ. ລະບົບນີ້ບໍ່ໄດ້ໃຫ້ບໍລິການແກ່ບຸກຄົນທົ່ວໄປ ແລະ ບໍ່ໄດ້ເກັບຂໍ້ມູນຈາກຜູ້ເຂົ້າຊົມ Facebook Page.

1. ຂໍ້ມູນທີ່ລະບົບເຂົ້າເຖິງ

  • Facebook Page Access Token — ໃຊ້ເພື່ອສ້າງໂພດ (ຮູບ+ຂໍ້ຄວາມ) ໄປຍັງ Page ທີ່ຜູ້ໃຊ້ອະນຸຍາດເທົ່ານັ້ນ. Token ຖືກເກັບໄວ້ໃນ Google Apps Script Properties (ສ່ວນຕົວ, ບໍ່ເປີດເຜີຍ) ບໍ່ຖືກສົ່ງໄປໃຫ້ບຸກຄົນທີສາມ.
  • ຮູບພາບສິນຄ້າ — ຈັດເກັບຢູ່ Google Drive ຂອງຜູ້ໃຊ້ເອງ ໃຊ້ເປັນວັດຖຸດິບສຳລັບການໂພດ.
  • ຂໍ້ຄວາມແຄບຊັ້ນ (Caption) — ບາງສ່ວນສ້າງໂດຍ AI (Google Gemini API) ຈາກຮູບພາບ, ບໍ່ມີການເກັບຂໍ້ມູນສ່ວນຕົວຂອງບຸກຄົນທີສາມ.

2. ຈຸດປະສົງການໃຊ້ຂໍ້ມູນ

ຂໍ້ມູນທັງໝົດຖືກໃຊ້ເພື່ອຈຸດປະສົງດຽວ: ຊ່ວຍໃຫ້ຜູ້ໃຊ້ໂພດເນື້ອຫາໄປຍັງ Facebook Page ຂອງຕົນເອງໄດ້ອັດຕະໂນມັດ. ພວກເຮົາບໍ່ຂາຍ, ບໍ່ແບ່ງປັນ, ແລະ ບໍ່ນຳຂໍ້ມູນໄປໃຊ້ເພື່ອຈຸດປະສົງອື່ນ.

3. ບໍລິການພາຍນອກທີ່ເຊື່ອມຕໍ່

  • Google Apps Script / Google Drive / Google Sheets — ຈັດເກັບຂໍ້ມູນ ແລະ ປະຫວັດການໂພດ
  • Google Gemini API — ສ້າງຂໍ້ຄວາມແຄບຊັ້ນ ແລະ (ໃນບາງກໍລະນີ) ຮູບພາບ
  • Meta Graph API (Facebook) — ໂພດເນື້ອຫາໄປຍັງ Page
  • LINE Messaging API — ແຈ້ງເຕືອນຜົນການໂພດ (ຖ້າເປີດໃຊ້)

ແຕ່ລະບໍລິການຂ້າງເທິງມີນະໂຍບາຍຄວາມເປັນສ່ວນຕົວຂອງຕົນເອງ ຊຶ່ງໃຊ້ຄວບຄູ່ກັບນະໂຍບາຍນີ້.

4. ການເກັບຮັກສາ ແລະ ຄວາມປອດໄພ

Token ແລະ ຂໍ້ມູນລັບທັງໝົດຖືກເກັບໄວ້ໃນ Script Properties ຂອງ Google Apps Script ຊຶ່ງເຂົ້າເຖິງໄດ້ສະເພາະເຈົ້າຂອງບັນຊີ Google ທີ່ສ້າງໂປຣເຈັກນີ້ເທົ່ານັ້ນ.

5. ການລຶບຂໍ້ມູນ

ຜູ້ໃຊ້ສາມາດລຶບຂໍ້ມູນທັງໝົດໄດ້ທຸກເວລາ ໂດຍການລຶບ Script Properties, Google Sheet, ຫຼື ຖອນການອະນຸຍາດ (Revoke) Facebook App ຈາກການຕັ້ງຄ່າບັນຊີ Facebook ຂອງຕົນເອງ.
